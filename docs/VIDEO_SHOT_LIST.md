## Pre-recording setup

- Run `npm run validate` and keep the successful output off-camera.
- Start fixture mode with `DATAHUB_MCP_MUTATIONS_ENABLED=false`.
- Open only ContextSeal, the synthetic DataHub target, and its decision document.
- Use a clean browser profile or hide the bookmarks bar and personal avatar.
- Enable Do Not Disturb; close email, messaging, payment, and account tabs.
- Confirm no `.env`, terminal, token screen, username path, or unrelated repository is visible.
- Put the DataHub proof caption on screen: **Disposable local DataHub · synthetic metadata · no production/customer data**.
